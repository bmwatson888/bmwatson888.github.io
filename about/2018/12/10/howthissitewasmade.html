    How this site was made…

    less than 1 minute read , Posted: December 10, 2018

    I’m not a big fan of hosting providers, so this site is run on static HTML and CSS pages, hosted within GitHub for free. However, the pages of this site are all dynamically generated using Jekyll, which is a Ruby-based application that converts flat template files into HTML pages.

    The site theme is a custom theme made specifically for Jekyll sites, called Minimal Mistakes, which takes a ton of the work out of creating the look-and-feel of the site, and managing layouts.

    A special shout-out to the creators of both Jekyll and Minimal Mistakes for making some awesome tools for us to use.

    This site took around 4-5 hours to setup, and most of that was just making minor tweaks.
